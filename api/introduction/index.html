阅读方式

该接口文档旨在对 Reactive Form 开放的 API 提供完整的输入、输出参考， 主要采用 JSDoc 风格对 API 进行描述。 在 JSDoc 基础上，为了行文清晰，文档按模块命名空间对 API 进行组织， 模块中挂载的实例或实例属性、方法，以 # 号起始作为标识。

另外，有源码阅读权限的读者需要注意，源码中暴露但未在此文档中体现的 API 有可能会被移除或产生较大变化。 因此，请以此文档作为主要的 API 参考。
